# 今晚做什么 PWA

这是一个面向个人使用的静态 PWA。所有日常数据默认保存在当前浏览器的 `localStorage` 中，不需要账号或数据库。

正式访问地址：<https://chenhonesty.github.io/home-evening-planner/>

## 已实现

- 依据精力、跳舞/加班、用餐状态生成晚间计划
- 跳舞与加班分别选择时长，夜宵可以与限时游戏并行
- 将 23:00 前的限时游戏作为硬约束
- 洗衣服拆为“放入并启动 5 分钟”和“取出晾晒 10 分钟”两个阶段
- 支持轻量 Todo 与鸡蛋、背包等快速模板
- 完成和延期计划，记录周期任务和整理项目
- 仅在做饭或使用碗筷后触发碗筷流程
- 使用 Open-Meteo 查询真实天气并推荐床品洗涤日期
- 家庭物品位置、库存状态、待购和家庭采购记录
- 离线应用壳、本机持久化、JSON 导入和导出
- iPhone 添加到主屏幕的安装说明

## 在电脑上测试

在本目录的上一级运行：

```bash
python3 -m http.server 8080 --directory outputs/home-evening-pwa
```

然后在电脑浏览器打开：

```text
http://localhost:8080
```

不要直接双击 `index.html`。直接打开文件时，本机保存可以工作，但 Service Worker 和离线安装不会启用。

## 在同一 Wi-Fi 的手机上临时测试

1. 在 Mac 上执行上面的 `http.server` 命令。
2. 执行 `ipconfig getifaddr en0` 查看 Mac 的局域网地址。
3. 在 iPhone Safari 打开 `http://局域网地址:8080`。

局域网 HTTP 适合测试页面和数据保存，但完整 PWA 安装与离线能力应使用 HTTPS。

## 正式安装到 iPhone

1. 将整个 `home-evening-pwa` 目录部署到任意支持 HTTPS 的静态网站托管服务。
2. 使用 iPhone Safari 打开部署地址。
3. 点击 Safari 的“分享”。
4. 选择“添加到主屏幕”。
5. 从手机桌面打开。

可以部署到 GitHub Pages、Cloudflare Pages、Netlify 或公司内部静态托管。部署时不需要构建命令，发布目录就是当前目录。

## 数据安全

- 数据不会自动上传到自建服务器。
- 查询天气时，用户输入的城市会发送给 Open-Meteo 的地理编码和天气接口。
- 换手机、清理 Safari 网站数据或更换部署域名之前，请在“回顾 → 数据与安装”中导出 JSON 备份。
- 导入备份会覆盖内存中的当前状态，建议先导出旧数据。

## 后续适合增加

- 定时推送需要服务端 Push 或与 iOS 快捷指令/提醒事项配合
- 云端同步需要登录和数据库
